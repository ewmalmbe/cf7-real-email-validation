== Description ==
An add-on for <a href="http://wordpress.org/extend/plugins/contact-form-7/">Contact Form 7</a> that valided an email field whether it is real or not using <a href="https://neverbounce.com/help/api/getting-started-with-the-api/" target="_blank">NeverBounce API</a>

Visit the [Contact Form 7 Real E-mail Validation](https://github.com/seekingcoder/cf7-real-email-validation) for additional information.

Get NeverBounce API key from [here](https://neverbounce.com/help/api/getting-started-with-the-api/)

= IMPORTANT NOTES: =
This plugin will work with Contact Form 7 4.1 or higher.

== Installation ==

1. Install using the Wordpress "Add Plugin" feature -- just search for "Contact Form 7 Real E-mail Validation".
1. Confirm that [Contact Form 7 v4.1+](https://wordpress.org/plugins/contact-form-7/) is installed and activated. Then activate this plugin.
1. Go to Never Bounce API page from WP Settings menu.
1. Update the settings with API credentials.
